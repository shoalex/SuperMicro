# IPMITool

The Project show how to connect and work with SuperMicro IPMI Tool and motherbard

# Features
- Connect to IPMI motherboard
- Control power remoutly
- Control fan speed


### Prerequisites

What things you need:
* Visual Studio
* IPMITool folder and the file SMCIPMITool.exe
